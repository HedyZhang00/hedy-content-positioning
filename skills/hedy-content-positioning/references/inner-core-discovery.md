# IP 内核与长期能量审计

## 何时启用

只在以下情况启用：候选方向难取舍、商业价值接近、差异化过弱、过去经常无法持续、看似赚钱的方向引发强烈抗拒、外部案例不足，或定位过于普通。

该模块不能替代案例、读者反馈和商业证据。

## 单问题审计

根据缺口选择问题，不必全部问完：

1. **原始驱动力**：没有要求和奖励时，用户仍会反复投入什么？16 岁以前是否已有相似行为？
2. **无意识优势**：什么事用户觉得自然，别人却觉得困难？必须追问一个真实事件。
3. **心流与低阻力输出**：什么事情容易忘记时间、做完更兴奋、素材会自然再生？
4. **表达冲动与愤怒点**：看到什么常见观点会忍不住反驳或重新解释？
5. **抗拒与嫉妒信号**：什么赚钱方式强烈不想做？最羡慕哪种创作者或生活，真正渴望的是什么？

## 使用边界

- 只写“观察到的信号”“可能的倾向”“待验证假设”。
- 不使用荣格、盖洛普或其他体系给出确定人格类型，除非用户提供了正式测评且只把它作为辅助证据。
- 一个方向商业证据强但长期消耗高时，明确写出代价，不自动选择市场最大方向。
- 一个方向能量高但没有市场证据时，只能作为低成本实验候选。
